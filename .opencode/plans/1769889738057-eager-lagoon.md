# Plan: Fork Watcher Landing Page

## Goal
Create a beautiful, bold, and colorful landing page for Fork Watcher (a GitHub repository forks viewer) at `src/app/page.tsx`.

## Design Decisions

### Visual Style
- **Bold & Colorful**: Vibrant gradients, strong visual impact
- **Modern gradient backgrounds** using purple-to-pink-to-blue gradients
- **Large typography** with dramatic sizing
- **Minimal structure**: Hero + CTA only (no footer, testimonials, etc.)
- **Dark mode optimized** with gradient adjustments

### Implementation Details

#### Gradient Animation CSS
```css

  0%, 100% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
}
```

- Primary gradient: `from-violet-600 via-fuchsia-500 to-cyan-500`
- Dark mode gradient: `from-violet-900 via-purple-800 to-indigo-900`
- Text: White on gradient, dark on light sections
- Accents: Pink (`fuchsia-500`), Cyan (`cyan-400`)
