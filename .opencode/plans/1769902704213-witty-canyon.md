## Design Document Summary

### Color Philosophy
Y2K Jungle Green - synthetic, digital, glowing:
- **Primary:** #39FF14 (acid green)
- **Backgrounds:** Dark green-tinted blacks
- **Accents:** Chrome/silver for contrast

### 3D Philosophy
- Only on interactive elements
- Layered shadows + subtle transforms
- No heavy skeuomorphism
- Smooth transitions (300ms ease-out)

### Nature Philosophy
- Calm, breathable spacing
- Rounded corners (not bubbly)
- Readable density
- No brutalist sharp edges

### Y2K Elements
- Window panels (fake OS aesthetic)
- Glowing accents
- Terminal/code blocks
- Sticker badges
- Gradient text
- Subtle scanlines/noise

## Additional Fix: Next.js 16 Dynamic Params

**Issue:** Next.js 16 changed `params` to be a Promise that must be awaited.

**Error:** `params.username` - params is a Promise and must be unwrapped with `await` or `React.use()`

**File to Fix:** `src/app/github/[username]/page.tsx`

**Changes Required:**

1. Update `PageProps` interface:
```typescript
interface PageProps {
  params: Promise<{ username: string }>;
}
```

2. Update `generateMetadata` to await params:
```typescript
export async function generateMetadata({ params }: PageProps) {
  const { username } = await params;
  return {
    title: `${username}'s GitHub Forks | Fork Watcher`,
    description: `View all repositories and forks for ${username} on GitHub`,
  };
}
```

3. Update `UserForksPage` to use `React.use()`:
```typescript
import { use } from 'react';

export default function UserForksPage({ params }: PageProps) {
  const { username } = use(params);
  const decodedUsername = decodeURIComponent(username);
  // ... rest of component
}
```

**Note:** The component cannot be async when using `use()`, so we need to use the React.use() hook instead.
